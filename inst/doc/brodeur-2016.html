These p-values come from the paper: Star Wars: The Empirics strike back and were downloaded from the authors website http://marcsangnier.free.fr/research_en.php on 2017-07-23. The file that is downloaded from the site is called brodeur_le_sangnier_zylberberg_replication.7z. The file was unzipped and in the main directory there was a file called final_stars_supp.dta. This is where this vignette begins.

2.3 Select the appropriate columns and clean

brodeur2016 = pdat %>% mutate(pvalue=2*(1-pnorm(t_stat_raw)),journal = journal_id) %>%
  mutate(field="economics", abstract=FALSE) %>%
  mutate(operator = NA, doi = NA, pmid=NA) %>%
  select(pvalue,year,journal,field,
         abstract,operator,doi,pmid) %>%
  filter(!is.na(pvalue))
